# newmaybe

个人写作站点，记录一些随笔、观察与思考。

网址：[newmaybe.com](https://newmaybe.com)

## 关于

「newmaybe」是一个刻意慢下来的地方——没有算法推荐，没有评论区，没有追踪代码。只有文字本身。

内容以中文为主，涵盖日常随笔、阅读感想与个人观察。文章会长期保留，不追求更新频率。

## 技术

静态站点，用 [Astro 6](https://astro.build) 生成，托管在 Cloudflare Pages。无客户端 JavaScript 框架，无第三方字体请求，无 Cookie。

```bash
npm install
npm run dev    # http://localhost:4321
npm run build
```
